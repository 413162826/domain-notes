Domain Notes 是一款"每个域名一本笔记"的浏览器扩展。当你在浏览某个网站时点击插件图标，它会自动加载该域名对应的笔记本，让你像便签一样快速记录信息——无论是 API 文档要点、阅读心得、工作笔记，还是临时备忘。

━━━━━━━━━━━━━━━━━━━━
🎯 为什么需要它？
━━━━━━━━━━━━━━━━━━━━

浏览资料时经常想"这里有用，先记一下"，但现有笔记软件要么太重、要么和当前页面割裂。Domain Notes 解决一件事：**让笔记与浏览同步发生**。

• 看到 github.com 的项目想记几句？点开就是 github.com 的笔记本
• 切到 docs.github.com 看文档？自动并入同一本笔记
• 切到 google.com 搜东西？又是另一本独立笔记

━━━━━━━━━━━━━━━━━━━━
✨ 核心特性
━━━━━━━━━━━━━━━━━━━━

📂 **智能域名归属（eTLD+1）**
不同子域、不同子页自动归入同一本笔记。docs.github.com、api.github.com 共享一本"github.com"笔记本。

⚡ **完整 Markdown 快捷键**
- `Ctrl+1~6`：H1~H6 标题
- `Ctrl+B/I`：粗体 / 斜体
- `Ctrl+L` / `Ctrl+Shift+L`：无序 / 有序列表
- `Ctrl+Shift+T`：待办清单
- `Ctrl+K`：插入链接
- `Ctrl+Shift+C`：代码块
- `Ctrl+/`：切换所有笔记视图

⌨️ **输入即触发**
- `# ` 空格 → 标题
- `- ` / `1. ` 空格 → 列表
- `> ` 空格 → 引用
- ``` ``` ``` → 代码块
- `**粗体**` `*斜体*` `---` 等通通支持

🖼️ **粘贴/拖拽图片**
直接 `Ctrl+V` 粘贴截图，自动转 base64 内嵌。点击图片可全屏预览。

🔗 **一键插入当前页面引用**
点击工具栏 🔗 按钮，自动把当前标签页的标题和 URL 作为引用块插入笔记，记下"这条笔记是看哪个具体子页时写的"。

🌙 **自动深色模式**
跟随系统主题切换，夜晚阅读不刺眼。

💾 **导出 / 导入备份**
所有数据一键导出为 JSON 文件，换电脑、做备份、迁移都方便。

🔒 **纯本地存储**
所有内容保存在浏览器本地存储区，**不上传、不联网、不追踪**。代码完全开源，可自行审计。
