Domain Notes is a "one notebook per website" browser extension. While browsing any site, click the extension icon to instantly open the notebook for the current domain — perfect for quick API notes, reading highlights, work memos, or any thought you want to capture without context-switching to a heavy notes app.

━━━━━━━━━━━━━━━━━━━━
🎯 Why this exists
━━━━━━━━━━━━━━━━━━━━

You're browsing a site and think "I should remember this." Existing note apps are either too heavyweight or disconnected from what you're reading. Domain Notes solves one thing: making notes happen exactly where you browse.

• Reading github.com? You get the github.com notebook.
• Jump to docs.github.com? Same notebook — auto-merged.
• Switch to google.com? Different notebook, automatically.

━━━━━━━━━━━━━━━━━━━━
✨ Features
━━━━━━━━━━━━━━━━━━━━

📂 **Smart domain grouping (eTLD+1)**
All subdomains and subpages of the same site share one notebook. docs.github.com and api.github.com both write into "github.com".

⚡ **Full Markdown shortcuts**
- `Ctrl+1~6` — Headings H1–H6
- `Ctrl+B/I` — Bold / italic
- `Ctrl+L` / `Ctrl+Shift+L` — Bullet / numbered lists
- `Ctrl+Shift+T` — Task list
- `Ctrl+K` — Insert link
- `Ctrl+Shift+C` — Code block
- `Ctrl+/` — Toggle "all notes" view

⌨️ **Input triggers**
Type `# `, `- `, `1. `, `> `, ``` ``` ```, `**bold**`, `*italic*`, `---` — they all auto-convert.

🖼️ **Paste & drop images**
`Ctrl+V` to embed screenshots as base64. Click any image for fullscreen preview.

🔗 **Insert current page reference**
One click inserts the current tab's title + URL as a quote block — perfect for tracking which subpage a note came from.

🌙 **Auto dark mode**
Follows your system theme.

💾 **Export / import backup**
Download all your notes as JSON. Move between machines, keep backups.

🔒 **100% local storage**
Everything stays in your browser's local storage. No uploads, no network, no tracking. Open source and auditable.
